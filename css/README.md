# Propriedades necessárias:
```css
width: propriedade que nos permite adicionar valores relacionados a largura de uma página ou elemento, bloco.
```
```css
height: propriedade que nos permite adicionar valores relacionados a altura de uma página ou elemento, bloco.
```
```css
display: propriedade que nos permite definir o modo como vamos exibir um determinado bloco ou elemento; block, quebra de linha (um debaixo do outro), inline (um do lado do outro), none (remove o elemento), inline-block (ele mantém um elemento do lado do outro, porém em comparação ao inline permite que a gente adicione largura/altura e espaçamentos no elemento. é um mix do block e do inline).
```
```css
margin: margem que define os espaçamentos externos entre um bloco e outro.
```
```css
padding: define os espaçamentos internos de um determinado bloco.
```
```css
color: define a cor do texto.
```
```css
background-color: define a cor de fundo.
```
```css
font-family: define a fonte do texto.
```
```css
font-size: define o tamanho da fonte.
```
```css
@media: A media query é uma regra que determina uma condicional para cada tipo de dispositivo. Então dependendo de qual resolução você poderá incluir regras específicas para ele

Para utilizarmos o @media podemos escrever de duas formas: usando o screen para definir o tipo de media ou apenas @media. 
```
```css
Pontos de quebra (breakpoints): são quando definimos em quais momentos o nosso site irá sofrer alterações visualmente. Quando o breakpoint é definido conseguimos dizer ao navegador que dependendo da forma como ele será visualizado poderá ter um visual ou comportamento diferente, através da @media.
```
